# NUST Volleyball Club Website

Join me on my journey as I built a website for my **University Volleyball club**(Still in construction) but we add on day by day till completion

![Project Demo](demo.gif)

## 🗺️ Site Map & Project Structure

The project is organized into a flat structure for easy navigation and deployment:

```text
├── index.html           # Home page with Match Countdown and news plus schedule section
├── schedule.html        # Match fixtures with category filtering
├── roster.html          # Team players with category filtering
├── gallery.html         # Photo collection of club events
├── history.html         # Club background and achievements
├── club info.html       # Club Management, contact and training details
├── activity.html        # Latest news and training feed
├── join.html            # Form for new members who want to join
├── styles.css           # Global styles
├── script.js            # Unified logic for timer & filtering
└── assets/              # Images, logos, and player photos
│    ├── logo.jpg
│    ├── news/
│    └── players/
│
└── data/
    ├── activities.js
    ├── executives.js
    ├── players.js
    ├── schedule.js
    └── timeline.js

```

## 🚀 Features
* **Live Countdown:** Dynamic JS timer for the next match kickoff.
* **Universal Filtering:** One-click filtering for Blazers, Falcons, and Female teams across both Roster and Schedule pages.
* **Semantic HTML:** Optimized for SEO and Accessibility (ARIA).

## 💻 Development Environment
* **OS:** MX Linux 25
* **Specs:** Intel Celeron N4500 | 8GB RAM
* **Tools:** Vanilla HTML5, CSS3 (Grid/Flexbox), ES6+ JavaScript
